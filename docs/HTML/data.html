Data

For the purpose of conducting our project, we refer to the data from Census.Gov that provides information on educational levels, educational institutions, and salaries in U.S. counties. First, educational levels represent the percentage of people above 25 years old with a Bachelor's degree. Then, from the "Firms" API, we were able to retrieve the number of educational centers per county. Lastly, we used an Excel file from the Economic Bureau to retrieve GDP per county, representative of our salary data. From the Census database, we have chosen two datasets – the Economic Census Survey (ECS) and the American Community Survey (ACS) – containing our variables of interest.

Data Sources

  • GDP:
    • For final data frame: .xlsx file from the BEA (Bureau of Economic Analysis), available here, set for 2017 and for GDP per Capita in each County. Called us_gdp_2009-18.
    • For Forecasting Data: .csv file from the World Bank, available here, GDP per capita from 1960 adjusted to 2015 inflation rate. Called gdp_us_over_time.csv.
    • For race analysis: .xlsx file from the US Department of Labor, available here , used only NATIONAL level data, and from a limited number of races for a general overview. Called gdp_per_race.xlsx
  • EDUCATION:
    • For final data frame: API query from the US Census Bureau, available here , set for Educational Firms and percentage of citizens with a Bachelor’s Degree.
    • For Forecasting Data: .xlsx file from the US Census Bureau, available here , set for all ages and sex. Called college_percentage.xlsx
    • For race analysis: .xlsx file from the US Census Bureau, available here , set for both sexes and diversifies by race. Called edu_perc_race.xlsx
  • UTILITY:
    • GEO_ID code to COUNTY/STATE name converter: .tsv and .xslx file from @ChuckConnel’s public GitHub repository, available here . Called fips2county.tsv

    • As indicators of working success, we decided to use GDP per capita (referred to as GDP p.c.), as it is an efficient measure of citizens' performance. While we acknowledge that money is not everything in life and other factors are important (e.g., job impact on the world, job satisfaction, popularity), they are subjective and hard to retrieve. Therefore, we will focus on GDP p.c. as a salary indicator, and for measuring higher education levels, we will use the Census Bureau data on College/Bachelor’s Degrees attainment, providing us with a basic understanding of educational attainment in the country.
